# Adivina-Numero-Logica-Programacion



## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un juego interactivo en Python donde el usuario debe adivinar un número aleatorio generado por el sistema entre 1 y 10.



El programa fue desarrollado aplicando los conocimientos adquiridos en la materia de Lógica de Programación, integrando estructuras condicionales, ciclos repetitivos, funciones y estructuras de datos.



---



##  Objetivo

Desarrollar un software funcional que permita al usuario jugar múltiples partidas, registrar intentos y visualizar estadísticas acumuladas.



---



##  Funcionalidades

\- Menú principal interactivo

\- Nueva partida

\- Generación de número aleatorio

\- Registro de intentos realizados

\- Mensajes de ayuda (mayor o menor)

\- Estadísticas acumuladas

\- Opción para salir del programa

\- Posibilidad de jugar varias veces



---



##  Estructuras utilizadas

En este proyecto se implementaron:



\- Funciones (modularización del código)

\- Archivo principal (main.py)

\- Archivo de funciones (funciones.py)

\- Ciclos while

\- Condicionales if / elif / else

\- Listas para almacenar intentos

\- Librería random

\- Variables acumuladoras para estadísticas



---



##  Estadísticas

El sistema registra:

\- Intentos por partida

\- Lista de números ingresados

\- Estadísticas acumuladas
